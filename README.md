Server-tan - Core
=================

Core classes and logic for Server-tan bot.

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'server_tan-core'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install server_tan-core
