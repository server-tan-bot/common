Server-tan - Common
===================

Common structures and libraries for Server-tan.

Installation
------------

Add this line to your application's Gemfile:

```ruby
gem 'server_tan-common'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install server_tan-common
